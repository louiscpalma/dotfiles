# dotfiles
Linux dotfiles and docker image with core workspace utilities

All configuration is intended to be run on a terminal with Solarized color theme, however the colors should transpose well enough to be usable with other terminal colors.

## vscode image
The `vscode` image uses [code-server](https://github.com/cdr/code-server) to deploy a Web UI of VS Code. The UI is pre-configured to use `Solarized Dark` theme and set tab width to 2 space.

### installed extensions
- vscodevim.vim
- ms-kubernetes-tools.vscode-kubernetes-tools
- redhat.vscode-yaml
